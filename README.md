# Attendance Tracking System (ATS)

## Overview
The Attendance Tracking System (ATS) is a terminal-based Python application developed for the PROG103 Principles of Structured Programming assignment.

The system is designed to help record and manage student attendance efficiently in an educational environment.

---

## Features
- Mark student attendance
- View attendance records
- Search student records
- Update attendance status
- Delete attendance records
- Display attendance summary
- Input validation for attendance status
- Menu-driven terminal interface

---

## Programming Concepts Used
This project demonstrates the application of:
- Variables and data types
- Functions
- Conditional statements
- Loops
- Lists and dictionaries
- Input validation
- Modular programming

---

- Python
- Visual Studio Code
- GitHub
